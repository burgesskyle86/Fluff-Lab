# Install Instructions

1. Download `fluff-lab-v4.0.zip`.
2. Unzip the file.
3. Open your GitHub repository.
4. Upload the contents of the unzipped folder.
5. Overwrite existing files.
6. Commit changes using:

Title:
Fluff Lab v4.0 - The Laboratory Update

Description:
Major update introducing the Laboratory foundation.

Added:
- Sticky Laboratory navigation
- Formula Summary screen
- Required Lab Supplies section
- Guided Experiment workflow
- Lab Supplies with current product defaults
- Kitchen and Metric measurement setting
- Interactive ingredient amount controls
- Product catalog using current pantry defaults
- Protocol naming and saved ingredient history
- Experiment Log with active and completed experiments

Changed:
- Formulas now use generic ingredient names instead of brand names.
- Siggi's Protein Yogurt is now the default yogurt.
- Ingredient amounts appear only after selecting Make This Formula.
- Experiments are created only after Make This Formula is selected.
- Protocols are saved from the exact ingredients/products used.

Improved:
- Mobile navigation
- Formula preview clarity
- Inventory structure, now called Lab Supplies
- Laboratory branding and terminology

7. Wait for GitHub Pages to deploy.
8. Open the app with `?v=4.0` at the end of the URL.
