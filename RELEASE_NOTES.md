# Fluff Lab v4.0 — The Laboratory Update

This release establishes the foundation for Fluff Lab as a food experimentation laboratory.

## Highlights

- Added Lab Supplies so formulas can stay generic while each user chooses their current products.
- Added a Formula Summary screen with Required Lab Supplies.
- Added Guided Experiment mode with adjustable ingredient amounts.
- Added Kitchen and Metric measurement options.
- Added sticky navigation so Back and Laboratory are always visible.
- Added Protocol naming and saved ingredient history.
- Updated default yogurt to Siggi's Protein Yogurt.

## Why we built this

Earlier versions stored formulas like recipes with specific products. v4.0 separates the formula from the user's actual products. This lets Fluff Lab support experimentation, product swaps, more accurate macros, and future features like inspiration, Lab Reports, and supply-based recommendations.
