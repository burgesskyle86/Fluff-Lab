Lab Manual

Getting Started

Start from Home, tap New Formula, choose a formula, review the base formula, add additives if desired, adjust ingredient amounts if needed, then Start Experiment. Amount/calorie/protein edits are recorded in the Experiment Log. After you mix and taste it, add observations and a conclusion, then Promote to Protocol if it is worth repeating.

Formula

A formula is the working version of a protein dessert. Ingredient amounts, calories, and protein can be edited directly before testing.

Additives

Additives are optional ingredients like fruit, graham crackers, PB2, juice, extracts, or sprinkles.

Observations

Use observations for what you noticed: texture, flavor, chill time, sweetness, thickness, and ratings.

Conclusion

The conclusion is the final lesson from the experiment. Write what you would repeat next time.

Protocols

A protocol is a formula that has been tested and promoted because you want to make it again. Star a protocol to mark it as a favorite.
